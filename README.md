# Sistema de Rehabilitación CNC con Monitoreo ECG

## Descripción
Sistema integral para rehabilitación física que combina:
- Control CNC para movimientos precisos de rehabilitación
- Monitoreo ECG en tiempo real con Arduino Nano
- Gestión completa de pacientes
- Análisis de progreso con gráficas
- Interfaz gráfica profesional

## Características Técnicas
- **Lenguaje**: Python 3.11+
- **Hardware**: Arduino Nano + Sensores ECG + Control CNC
- **Librerías**: pygame, pyserial, numpy, pandas, matplotlib
- **Arquitectura**: Modular con clases especializadas

## Estructura del Proyecto
- `programatesis9.py` - Programa principal
- `backups/` - Datos de sesiones de pacientes
- `configuraciones/` - Archivos de configuración

## Funcionalidades Principales
1. **Gestión de Pacientes**: Registro, búsqueda y seguimiento
2. **Captura ECG**: Monitoreo en tiempo real durante ejercicios
3. **Control CNC**: Rutinas automáticas de rehabilitación
4. **Análisis**: Gráficas de progreso y comparación de sesiones
5. **Exportación**: Reportes para análisis médico

## Instalación
```bash
pip install pygame pyserial numpy pandas matplotlib
```

## Uso
```bash
python programatesis9.py
```
